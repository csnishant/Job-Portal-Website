# Job Portal Website 🚀

**Job Portal Website** is a full-stack web application built with the **MERN stack** (MongoDB, Express, React, Node.js).  
It allows users to browse job listings, apply for jobs, and provides basic authentication and CRUD features for job postings.

## 📌 About

This project is a **Job Portal** where job seekers can search and apply to available jobs, and recruiters can post/manage job listings.  
It uses modern technologies to deliver a fast and responsive user experience. :contentReference[oaicite:1]{index=1}

## 🧰 Tech Stack

| Component | Technology |
|-----------|-------------|
| Frontend  | React + Vite |
| Backend   | Node.js + Express |
| Database  | MongoDB |
| Authentication | JWT |
| UI        | Tailwind CSS / Custom CSS |
| HTTP Client | Axios |

---

## ⭐ Features

- User Sign up & Login (JWT Authentication)
- Browse job listings
- Apply for jobs
- API routes for job management
- Clean and responsive UI
